# vue-blog-admin

博客前台：[vue-blog](https://github.com/linfeng-weng/vue-blog)
博客后端：[vue-blog-server](https://github.com/linfeng-weng/vue-blog-server)

## 技术栈

- `Vue3` `Vue-Router` `Pinia` `Axios`
- 脚手架： `Vite`
- 富文本编辑器：`WangEditor`
- 组件库： `Element-Plus`

## 安装

1.安装依赖

```
    $ npm install
```

2.运行服务

```
    $ npm run dev
```
